How This Backtest Works

We replay every game of the 2026 season chronologically. For each game, we compute a prediction using only data available before that game. Accumulators (runs scored/allowed, win-loss records) update after each prediction, never before. Bradley-Terry ratings refit every 100 games.

This is a "walk-forward" backtest — the gold standard for time-series prediction validation. No peeking at future results.

Note: The PEAR comparison uses current ELO ratings (not point-in-time snapshots), so PEAR's accuracy may be slightly inflated. Our model's accuracy is computed honestly via walk-forward.
