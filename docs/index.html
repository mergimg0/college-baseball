How It Works

College Baseball Predictions uses a three-layer prediction model to generate calibrated win probabilities for every NCAA D1 baseball game.

Layer 1: Pythagenport Expected Win% with variable exponent calibrated for college baseball (2.13, not MLB's 1.83).

Layer 2: Dynamic Bradley-Terry ratings via MM algorithm with temporal decay.

Layer 3: Log5 matchup probability with home-field adjustment (~53% baseline).

Predictions refresh every 6 hours. Ratings update as new results come in, so predictions for future games improve over time.
