How It Works

ForgeStream Baseball uses a three-layer prediction model to generate calibrated win probabilities for every NCAA D1 baseball game.

Layer 1: Pythagenport Expected Win% with variable exponent calibrated for college baseball (2.13, not MLB's 1.83).

Layer 2: Dynamic Bradley-Terry ratings via MM algorithm with temporal decay.

Layer 3: Log5 matchup probability with home-field adjustment (~53% baseline).

Technical Details

Data sources: NCAA Scoreboard API, PEAR Ratings API, The Odds API (baseball_ncaa)
Refresh: Daily automated pipeline — scrape → rate → predict → publish
Coverage: 284 D1 teams, ~8,400 games/season
Stack: Python 3.12, SQLite, static HTML. No ML frameworks.
Odds: The Odds API — moneyline, run line, totals from 40+ bookmakers. Free tier.
Backtest: Chronological replay with Brier score calibration. No future data leakage.

No neural networks. The accuracy comes from correct modeling of college baseball's unique run environment, not from complexity.
